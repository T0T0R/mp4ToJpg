# mp4ToJpg
Extract all frames of a video file (mp4 or avi) into a folder.

## How to use the script
1. Download the `mp4ToJpg.py` Python script.
2. This script uses `av` and `pillow`, so make sure these packages are installed:
   - In your terminal, run `pip3 install av pillow`.
3. Launch the script with `.\mp4ToJpg.py myVideoPath` by replacing `myVideoPath` by the path of your video. Multiple videos can be passed and analyzed successively with `.\mp4ToJpg.py myVideo1Path myVideo2Path`.

Alternatively, the videos can just be drag-and-dropped onto the script file.
